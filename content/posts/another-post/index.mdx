## ES6 e além

Com a introdução do ES6 (ECMAScript 2015), JavaScript ganhou muitas novas características...

### Arrow Functions

Arrow functions permitem uma sintaxe mais curta para escrever funções...

```javascript
const add = (a, b) => a + b;
```


### Promises e Async/Await
Promises e async/await são abstrações que nos ajudam a lidar com operações assíncronas...

```javascript
const fetchData = async () => {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  console.log(data);
};
```
